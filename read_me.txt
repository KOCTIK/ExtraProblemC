
Extra problem C – “Skoczki”.

Using a mouse choose coordinates of starting and ending horse's positions, click on and the program finds the shortest way 

The program was written using a programming language JavaScript.

The “inputData.js” file contains an array of numbers.

The “algorithm.js” file contains the part of code, which creates a set of nodes and looking for a shortest way from start to finish.

The folders “css” and “js” contain files, which are needed for visualization.
